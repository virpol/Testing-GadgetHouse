# QA Documentation - GadgetHouse Store

This repository contains QA documentation for the **Online Electronics Store** project.  
The platform sells **mobile phones, laptops, headphones, and other electronics**.

## Structure
- **1_Test_Cases/** - test cases for design and functionality
- **2_Bug_Reports/** - detailed bug descriptions by task (DV and MV)
- **3_Notes/** - common QA notes, test data, and useful links

## Test Environment
- **Frontend**: React + responsive design
- **Backend**: Spring Boot (Docker)
- **Database**: PostgreSQL
- **URL**: `https://gadget-house-tc.netlify.app/`
